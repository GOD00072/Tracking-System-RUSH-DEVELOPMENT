## API Integration

### Endpoints Used:
```javascript
// Get items for an order
GET /api/v1/order-items?orderId={orderId}

// Create new item
POST /api/v1/order-items
Body: { orderId, productCode, priceYen, ... }

// Update item
PATCH /api/v1/order-items/{id}
Body: { itemStatus, trackingNumber, ... }

// Delete item
DELETE /api/v1/order-items/{id}
```

### Authentication:
- All endpoints require **Admin authentication** (`admin_token` cookie)
- Handled automatically by `api` instance with `authenticateAdmin` middleware
