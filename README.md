# Ship Tracking System

## Overview
ระบบติดตามสินค้าและบริหารจัดการคำสั่งซื้อแบบครบวงจร สำหรับการขนส่งทางเรือและทางอากาศ

## Technology Stack

### Frontend
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: TanStack Query + Zustand
- **Forms**: React Hook Form + Zod
- **Maps**: React Leaflet
- **Charts**: Recharts
- **Hosting**: Vercel

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL (Supabase)
- **Authentication**: Supabase Auth + Google Sign-In
- **Hosting**: Render

### Integrations
- **Data Management**: Airtable
- **Notifications**: LINE Official Account API
- **Accounting**: FlowAccount API
